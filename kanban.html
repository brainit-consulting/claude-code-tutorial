How to Use Kanban Board

Board Title

Click the title at the top to edit it. Press Enter or click away to save.

Managing Tasks

  • Add Task: Click "+ Add Task" at the top of any column
  • Edit Task: Click the pencil icon (✎) on a task
  • Delete Task: Click the trash icon (🗑) on a task
  • Move Task: Drag and drop anywhere in a column (highlighted when valid)

Task Numbers

Each task gets a unique number (#001, #002, etc.) for easy reference.

Build Button

Click the hammer icon (⚒) to copy a prompt to clipboard. Paste it into Claude Code to implement the task.

Markdown Export

  • Link File: Click "Link MD File" to connect a markdown file
  • Auto-save: File updates on create, edit, delete, or move
  • Reconnect: Click orange "Reconnect" button after page refresh

Done Column

  • Collapse/Expand: Click the arrow (‚ñº) to hide/show done tasks
  • Archive: Click "Archive All Done Tasks" to move completed tasks to archive
  • Restore: Click "Restore" on archived tasks to bring them back
  • Clear Archive: Click "Clear All" to permanently delete archived tasks
