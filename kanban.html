How to Use Kanban Board

Managing Tasks

  • Add Task: Click "+ Add Task" in any column
  • Edit Task: Click the pencil icon (✎) on a task
  • Delete Task: Click the X button on a task
  • Move Task: Drag and drop between columns

Task Numbers

Each task gets a unique number (#001, #002, etc.) for easy reference.

Build Button

Click the hammer icon (⚒) to copy a prompt to clipboard. Paste it into Claude Code to implement the task.

Markdown Export

Click "Link MD File" to connect a markdown file. It auto-updates whenever you create, edit, delete, or move tasks.
